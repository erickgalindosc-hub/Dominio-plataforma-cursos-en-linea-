# 📚 Proyecto Semana 03: Calculadora de Dominio

## 🎯 Dominio
**Plataforma de cursos online**

---

## 🧠 Descripción

Este proyecto simula el funcionamiento básico de una plataforma de cursos en línea.  
La idea principal es calcular los ingresos generados a partir de la venta de cursos básicos y premium, aplicando descuentos, comisiones y validaciones.

Se utilizan distintos tipos de operadores de JavaScript para representar situaciones reales del negocio.

---

## ⚙️ Funcionalidades

- Cálculo de ingresos por tipo de curso
- Cálculo de ingresos totales
- Promedio de ingresos por estudiante
- Aplicación de descuentos promocionales
- Cálculo de comisión de la plataforma
- Validaciones de condiciones (número de estudiantes, ingresos, etc.)
- Evaluación de reglas de negocio con operadores lógicos

---

## 🧮 Operadores utilizados

### ➕ Aritméticos
Se usan para calcular ingresos, promedios y totales:
- `+`
- `-`
- `*`
- `/`

### 🔁 Asignación compuesta
Para actualizar valores sin crear nuevas variables:
- `-=`

### 🔍 Comparación estricta
Para validar condiciones correctamente:
- `===`
- `>`

### 🔗 Operadores lógicos
Para combinar condiciones:
- `&&`
- `||`
- `!`

---

## 💡 Ejemplo de lógica aplicada

- Se calcula el total de ingresos sumando cursos básicos y premium
- Se aplica un descuento del 10%
- Luego se descuenta la comisión del 15% de la plataforma
- Se evalúa si el ingreso es alto
- Se verifica si hay suficientes estudiantes
- Se determina si se cumple un bono por rendimiento
