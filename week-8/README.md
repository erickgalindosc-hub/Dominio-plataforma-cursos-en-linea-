# 📦 Proyecto Semana 8 — Gestión de Inventario con Arrays

## 🎓 Dominio: Plataforma de Cursos Online

---

## 📌 Descripción

Este proyecto implementa un sistema de gestión de inventario utilizando **arrays en JavaScript**, aplicado al dominio de una **plataforma de cursos en línea**.

El sistema permite administrar cursos, realizar operaciones sobre el inventario y generar un reporte completo usando diferentes métodos de arrays.

---

## 🎯 Objetivo

Aplicar los principales métodos de arrays para:

- Gestionar datos de cursos
- Realizar operaciones de mutación
- Buscar y filtrar información
- Transformar datos
- Generar reportes claros en consola

## 📊 Datos del Dominio

El sistema trabaja con un array de cursos. Cada curso contiene:

- `id` → identificador único
- `name` → nombre del curso
- `category` → categoría
- `price` → precio del curso
- `active` → estado (activo o inactivo)
- `students` → número de estudiantes inscritos

### Ejemplo:

```js
{
  id: 1,
  name: "JavaScript",
  category: "programación",
  price: 50,
  active: true,
  students: 120
}
