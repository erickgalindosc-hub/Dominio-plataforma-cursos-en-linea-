# 🚀 Proyecto Semana 07 — Librería de Funciones

## 🎓 Dominio: Plataforma de Cursos en Línea

---

## 📌 Descripción

Este proyecto implementa una **librería de funciones reutilizables** aplicada al dominio de una **plataforma de cursos online**.  

El sistema permite gestionar información de cursos, incluyendo su nombre, categoría, precio y estado (activo/inactivo), además de generar un **reporte automatizado** usando funciones.

---

## 🎯 Objetivo

Desarrollar funciones reutilizables que:

- Procesen datos del dominio
- Devuelvan resultados mediante `return`
- Sean independientes del flujo principal del programa
- Se integren en un reporte final
